Project Overview

Smart Laptop Advisor was built as a capstone project. Its core regression engine estimates fair market value by training on real configurations listings. By evaluating combinations of processor clocks, GPU tiers, memory channels, and pixel density factors, the XGBoost regressor minimizes price deviations on testing splits to achieve high prediction accuracy.

Timeline

Development workflow

1

Data Scraping & Collection

Gathered real-world listings for 1,303 laptop configurations spanning companies, build parameters, and retail prices in INR.

2

Cleaning & Feature Engineering

Parsed storage capacity characters into numerical categories, extracted screen sizes, computed display PPI values from pixel dimensions, and isolated CPU and GPU brand keywords.

3

Model Candidate Benchmarking

Trained three regression models side-by-side on uniform train/test splits. Compiled MAE, RMSE, and R² scores to benchmark candidacy.

4

Hyperparameter Grid Search Tuning

Ran randomized cross-validation grids on XGBoost params (estimators depth, learning rates, column sub-samples) to maximize generalizability and achieve a champion R² score of 0.9182.

5

Workspace Deployment Layout

Wrapped the pricing pipeline, BI dashboards, workload matches, and comparison sheets into this high-end interactive SaaS advisor.
